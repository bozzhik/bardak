- [x] Next.js 16 (apps/web) с базовым layout
- [x] Convex подключён, демо-запрос работает
- [x] shadcn/ui установлен (50+ компонентов)
- [x] Docker (web + bot) + docker-compose
- [x] ESLint + TypeScript shared configs
- [x] Yandex Metrika

## После MVP (не сейчас)

> Эти фичи **не входят в текущий скоуп**. Записываем, чтобы не забыть.

- Память бота («запомни, что...»)
- «Душа бота» — кастомный стиль общения
- Граф-карта тегов (визуализация связей, как в Obsidian)
- AI-преобразования: перевод, переписывание, план
- Командные сценарии (группа с ботом)
- Голосовой диалог как интерфейс
- Напоминания на основе голосовых
